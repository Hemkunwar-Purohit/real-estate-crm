🏠 RealEstate CRM — Documentation

Version 1.0.0   Laravel 12   PHP 8.2+

📋 Requirements

  • PHP >= 8.2
  • MySQL >= 5.7
  • Composer >= 2.x
  • Node.js >= 18.x

⚡ Installation Steps

Step 1 — Extract & Setup

cd /your/htdocs/folder
# Extract the ZIP file
cd real-estate-crm

Step 2 — Install Dependencies

composer install
npm install
npm run build

Step 3 — Environment Setup

cp .env.example .env
php artisan key:generate

Edit .env file:

APP_NAME="RealEstateCRM"
APP_URL=http://localhost

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=realestate_crm
DB_USERNAME=root
DB_PASSWORD=

Step 4 — Database Setup

php artisan migrate
php artisan db:seed
php artisan storage:link

Step 5 — Run

php artisan serve

Open: http://localhost:8000

🚀 Features

  • Lead Management — Multiple sources (MagicBricks, 99acres, Housing.com)
  • Client Management — Buyer, Seller, Tenant, Landlord
  • Property Listings — RERA number, price, location
  • Deal Pipeline — Kanban board with drag & drop
  • Site Visits — Schedule, track, feedback
  • Reports — Charts, analytics, Excel export
  • User Management — Role-based access
  • Activity Log — All changes tracked

⚙️ Configuration

All settings in config/crm.php:

// Pipeline stages customize karo
'pipeline_stages' => [
    'new'         => 'New Inquiry',
    'site_visit'  => 'Site Visit',
    ...
],

// Lead sources add karo
'lead_sources' => [...],

// Currencies
