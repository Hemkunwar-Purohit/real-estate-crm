# Changelog

All notable changes to RealEstate CRM will be documented here.

## [1.0.0] - 2025-05-01

### Added
- Lead management with multiple source tracking
- Client management (Buyer, Seller, Tenant, Landlord)
- Property listings with RERA number support
- Deal pipeline with Kanban board (drag & drop)
- Site visit scheduling and tracking
- Dashboard with charts and analytics
- Reports with Excel export
- User management with role-based access
- Activity log for all changes
- Demo data seeder
